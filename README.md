# Jerick TM - School Management System

Jerick TM is a modern, secure, and responsive school management web application designed to handle:

- **Tasks & Assignments** – Create, edit, delete, and track tasks.
- **Grades & Scores** – Record and view student grades by subject.
- **Events & Calendar** – Manage school events, deadlines, and reminders.
- **User Profiles** – Student, Teacher, and Administrator roles.
- **Notifications** – Alerts for overdue tasks and upcoming events.
- **Role-Based Access** – Only authorized users can access certain features.

## Features

- Modern dashboard with summary of tasks, grades, and events.
- Dark mode and light mode toggle.
- Interactive tables for tasks and grades.
- Secure login system with password recovery.
- Activity logs for administrators.
- Fully responsive and browser-compatible.

## Tech Stack

- **Frontend:** React or Vue.js + TailwindCSS / Material UI
- **Backend:** Node.js + Express
- **Database:** MySQL or PostgreSQL
- **Authentication:** JWT + bcrypt
